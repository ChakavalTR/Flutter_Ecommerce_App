# Flutter E-Commerce App

A modern E-Commerce mobile application built with Flutter and GetX, featuring a clean UI, product browsing, shopping cart management, favorites, and checkout functionality.

## Features

### Authentication

- User Sign In
- User Sign Up
- Forgot Password
- Session Management

### Home Screen

- Promotional Banner Slider
- Product Categories
- Flash Sale Section
- Best Selling Products
- Product Search

### Product Management

- Product Listing
- Product Details
- Product Images Gallery
- Product Variants (Color & Storage)
- Product Ratings & Reviews

### Shopping Cart

- Add to Cart
- Remove from Cart
- Quantity Management
- Order Summary

### Favorites

- Add to Favorites
- Remove from Favorites
- Favorite Products Screen

### Checkout

- Shipping Address
- Payment Method Selection
- Order Summary
- Order Confirmation

### Profile

- User Profile
- Edit Profile
- Order History
- Settings
- Logout
